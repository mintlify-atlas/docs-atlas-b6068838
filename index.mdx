## Quick Start

Get up and running in minutes with our simple installation process.

<Steps>
  <Step title="Install Dependencies">
    Install the required Python packages using pip:
    
    ```bash
    pip install curl_cffi tls_client prettytable
    ```
  </Step>
  
  <Step title="Configure Your Session">
    Set up your API credentials and proxy configuration:
    
    ```python
    from bet365 import Bet365AndroidSession
    
    session = Bet365AndroidSession(
        api_url="your-api-url",
        api_key="your-api-key",
        proxy="http://proxy:port",  # optional
        host="www.bet365.com"
    )
    ```
  </Step>
  
  <Step title="Start Scraping">
    Extract sports data and odds information:
    
    ```python
    # Navigate to homepage
    session.go_homepage()
    
    # Get available sports
    sports = session.extract_available_sports()
    
    # Fetch specific sport data
    soccer = next(filter(lambda m: m.name == "Soccer", sports))
    session.get_sport_homepage(soccer)
    ```
  </Step>
</Steps>

<Warning>
This SDK is designed to help developers handle bot-detection challenges and maintain stable access when interacting with Bet365 under legitimate and authorized use cases. It provides a controlled environment for managing request patterns and mitigating false positives from automated protection systems.
